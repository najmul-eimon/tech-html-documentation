overview

Techverse - IT Business HTML Template

Techverse is a HTML Template which is a high-Quality and well organized HTML5 Template specially designed to fit all the needs of IT Business and solutions. It will also be suitable for any kind of professional business template. It can be used for various business sites like startup, IT farm based service etc.

In this template has comments on each section included in the template so you can easily adapt your templates to suit your needs. This html template can be easily customizable. You can easily replace any picture with your own in just a few clicks.

It’s built with the most modern tools Tailwind CSS. it’s a very complete and highly customizable template that will help you build your own website layouts. It’s Simple, elegant, extensive, professional.

Change Site Title and Favicon

Go to every Html file (ex: index.html) to change the title and fav icon in <head> tag.

Also replace the favicon provided in /src/images/favicon.png file with the new one.

Change Logo

Go to every Html file (ex: index.html) and change logo in <header> tag.

Customize Menu

Go to every Html file (ex: index.html) and change menu items in <header> tag.

Customize Footer

Go to every Html file (ex: index.html) and change footer logo, social media, footer data in <footer> tag.

Change Copyright Infomartaion

Go to every Html file (ex: index.html) and change copyright information in <footer> tag.

Section Data

Go to every Html file (ex: index.html) and here you will find every section in <section> tag with detailed comment.

service Data

Go to /pages/service.html file. Here you will fild all the service sections. Or you can search by the classname "all-service" to find this section.

Blog Data

Go to /pages/blog.html to find all the blog sections. Or you can find by the classname "blog" to find this section.

Social Networks

Go to every Html file (ex: index.html) to change social links in <footer> tag.

Typography - Font Family

Go to /main.css and /tailwind.config.js file and put your font family link to change the font of the website.

Typography - Color

Go to /tailwind.config.js file and put your color code to change font color and other colors.
